📖 Homework Writeup

I explored the HTML video element. I found that there's a "controls" attribute that allows people to control the video, e.g., play, pause, and set volumes. I also found that I can add text between the video tags in case the video cannot be played. This text will only be displayed when the browser does not support the video tags.

I also used CSS animation and applied it to my H1 header. I learned that I need to write the @keyframes rules for the animation and provide them with a name. Then, I used this name to link the animation to the corresponding elements. An important thing to note is that I must specify the animation-duration; otherwise, the animation will not occur since the default duration is 0.

I completed the first and the second extra credit tasks. For the first task, I added an interactive mouse trailing effect based on the mouse position using the mousemove event. For the second task, I added meta elements to allow website previews on different social media. Based on the reading material, I utilized the open graph tags since they support the most social media.
